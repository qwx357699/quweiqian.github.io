# [作业讲解]算术运算符

```js
    <script>
        var C = 27.5;
        var F = 9 / 5 * C + 32;
        console.log("摄氏温度为" + C + "度，华氏温度为" + F + "度");
    </script>
```

```js
    <script>
        var money = 100000,
            month = 3,
            rate = 5.6; //利率百分比
        var earnMoney = money * rate / 100 / 12 * month;
        console.log("本金:" + money + ", 存储月数：" + month + ", 年利率:" + rate + ", 赚取利息：" + earnMoney);
    </script>
```
